# Elite Rail - Requirements

## Prerequisites
To run this project, you need to have Node.js and npm installed on your system.
- Node.js: v18.x or higher
- npm: v9.x or higher

## Core Backend Dependencies
These are required for the Express.js server:
- express == 5.2.1
- cors == 2.8.6
- dotenv == 17.4.2
- sqlite3 == 6.0.1
- nodemon == 3.1.14

## Core Frontend Dependencies
These are required for the React application:
- react == 19.2.6
- react-dom == 19.2.6
- react-router-dom == 7.15.0
- axios == 1.16.1
- lucide-react == 1.14.0
- framer-motion == 12.38.0
- vite == 8.0.12

## Development Utilities
- concurrently == 8.2.2

## Installation & Running
1. Install root dependencies:
   npm install

2. Install backend dependencies:
   cd backend && npm install

3. Install frontend dependencies:
   cd frontend && npm install

4. Run the application (starts both frontend and backend):
   npm run dev
